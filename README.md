# AI Business Analyst Dashboard

## Overview

AI Business Analyst Dashboard is an end-to-end Business Intelligence project that analyzes retail sales data using SQL Server, Python, Power BI, and Streamlit. The application provides interactive dashboards, KPI tracking, business analytics, and AI-ready insights to support data-driven decision-making.



## Tech Stack

* Python
* SQL Server (SSMS)
* SQL
* Power BI
* Streamlit
* Pandas
* Matplotlib
* Microsoft Excel
* Git & GitHub



## Features

* Total Sales KPI
* Total Profit KPI
* Total Orders
* Total Customers
* Sales by Category
* Profit by Region
* Monthly Sales Trend
* Top Customers Analysis
* Discount vs Profit Analysis
* Interactive Streamlit Dashboard
* AI-ready Business Insight Generator
